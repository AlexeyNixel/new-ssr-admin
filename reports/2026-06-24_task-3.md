## 3. События — создание на несколько дней

**Проблема:** для повторяющегося события нужно было каждый раз заново заполнять всю форму.

### Файл: `app/components/Modals/AdminEvent.vue`

**Решение:** диапазон дат — включаешь переключатель, выбираешь конечную дату, нажимаешь
один раз — создаются все события за каждый день диапазона.

- Переключатель «Создать на несколько дней» (показывается только в режиме создания)
- При включении появляется поле «Дата окончания» (`type="date"`) с `min` = дата начала
  и CSS-анимацией появления
- Счётчик в реальном времени: «Будет создано N событий»
- Кнопка меняет текст: «Создать событие» → «Создать N событий»
- При сабмите: последовательный цикл POST-запросов с одинаковым временем, но разными датами
- Toast по завершении: «Создано N событий»
- Исправлен старый баг: `eventTime` мутировался прямо в ref (`:00.000Z` добавлялось каждый
  вызов) — теперь строится в отдельный `payload`
- Добавлен `isLoading` со спиннером на кнопке
